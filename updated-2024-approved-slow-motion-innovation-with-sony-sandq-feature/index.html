In answering this question, its essential to understand the difference between color correction and color grading. Both editing techniques have vital differentiating characteristics, so its wrong to use them interchangeably.

Color correction ensures that videos look as natural to the human eye as possible. If theres a technical color error, for example, due to bad lighting, color correction helps fix that.

However, the aim of color grading isnt fixing color mistakes but using colors to convey a mood, atmosphere, or style. Some videographers classify color correction as the first phase in the color grading process.

Generally, you can use LumaFusion to color correct your footage. However, you cant color grade with the app except you decide to tamper with the color presets tabs color sliders. Even at that, you wouldnt get the perfect results you want.
