What is an SRT File?

An SRT file is a file that contains the content on a video in text format as well as the timing of the words and the order in which they show up on the video. It is one of the most popular captioning file formats in use today.

The SRT file only contains the subtitle information and must be used in combination with the video to be effective. When it is displayed correctly and loaded on a compatible video platform an SRT file can actually increase the effectiveness of the videos content.
