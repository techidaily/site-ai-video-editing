Step 2. Setup Screen Recording

You’ll be able to choose the computer screen display you wish to capture in the following window. If you just have one monitor, make that the default. OBS Studios should show you the screen in the window after a valid display has been chosen. We need to change your graphics settings if OBS is unable to successfully load the screen video. Click the Start Recording button in the bottom right corner when you’re ready to begin recording. When you’re finished, click Stop Recording, and a movie will be saved to the folder you chose in the preferences.
