The .apng (Animated Portable Network Graphics) extension is an unofficial extension to the Portable Network Graphics format (PNG ). It is a series of frames (a PNG image) that represent an animation sequence, producing a visual effect comparable to a GIF. APNG files support 24-bit pictures with 8-bit transparency. You can convert non-animated GIF files to APNG format. APNG files have the same.png extension as PNG files and can be opened by software like Mozilla Firefox, Chrome with APNG compatibility, and iOS 10s iMessage apps.
